## Project Overview

A full-stack web application with Netflix-inspired UI featuring user authentication, built using React and Node.js.

---

## Technology Stack

**Frontend:**
- React 19 + Vite
- React Router
- CSS3

**Backend:**
- Node.js
- Express.js

**Deployment:**
- GitHub Pages
- Vercel

---

## Features

1. **Responsive Login Page** - Works on all devices
2. **User Authentication** - Secure login system
3. **Protected Dashboard** - Accessible after login
4. **Modern UI** - Clean, professional design

---

## Technical Implementation

### Frontend
- React components with hooks (useState, useNavigate)
- Form validation and error handling
- Axios for API calls
- Custom CSS styling

### Backend
- Express server with REST API
- POST `/api/login` endpoint
- Input validation
- CORS enabled
